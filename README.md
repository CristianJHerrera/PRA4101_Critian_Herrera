# Portafolio de la asignatura Programacion Aplicada a la Ciencia de Datos

¡Hola! Bienvenido a mi repositorio personal. 

Este espacio está destinado a organizar, documentar y almacenar todo mi trabajo práctico, código y avances desarrollados a lo largo de este curso universitario. 

## Estructura del Repositorio

Para mantener un orden adecuado, los archivos están divididos en las siguientes carpetas principales:

* **`Actividades`**: Contiene las prácticas, ejercicios de laboratorio y dinámicas cortas realizadas en el transcurso de las clases.
* **`Tareas`**: Aquí se encuentran las asignaciones formales y deberes dejados por el docente para resolver fuera del horario de clases.
* **`Proyectos`**: Espacio dedicado a los trabajos más extensos y complejos, integrando múltiples conceptos del curso, como desarrollos web o análisis de datos.

## Notas Adicionales
Este repositorio se actualizará constantemente a medida que avancemos en el plan de estudios y se entreguen nuevas evaluaciones.
